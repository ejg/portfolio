Some of my latest projects

Mastering the Bluebook: Interactive Exercises

Mastering the Bluebook: Interactive Exercises is an e-Learning application for law students and others to master legal citation through a series of interactive exercises.

For this project, I

  • developed a reusable jQuery framework for all the various interactive exercises
  • built an instructor dashboard to create classes and tests, view student progress and display student, class, and school reports.
  • used HTML5, CSS, jQuery, PHP and mySQL

This project was developed in collaboration with Firestream Media and is available for purchase at Cap Law.
